== Description ==
An SEO friendly widget the create and display user profile card.



You can add the following: 
Name
Profile Image
Header Cover
Tagline
Description
Social media links:(facebook,twitter,website,linkedIn)

with schema.org microdata for SEO.

You can also modify some Background Color and Font  Color.


This plugin is still under development. You can visit <a href="http://users-profile-card-widget.22bees.com/">Users Profile Card Widget Demo</a>.
